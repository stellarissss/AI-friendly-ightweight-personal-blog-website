# 网站前端黑客风格改造计划

## 一、项目现状分析

## 二、设计方向

### 美学风格：黑客终端美学（Cyber Terminal）
- **主色调**：深黑背景 (#0a0a0a) + 亮白前景 (#fafafa) + 霓虹青色点缀 (#00fff9) + 品红高光 (#ff00c1)
- **字体**：JetBrains Mono 等宽字体为主，营造终端/代码编辑器氛围
- **视觉元素**：
  - 故障艺术（Glitch）文字效果
  - CRT扫描线质感
  - 终端命令行风格的UI组件
  - ASCII艺术装饰
  - 打字机动画效果

### 互动背景：粒子网络系统（Particle Network）
**不使用代码雨**，设计一个可交互的粒子物理系统：
- 屏幕上分布多个发光粒子点
- 粒子间距离较近时自动连线，形成网络结构
- **鼠标交互**：
  - 鼠标移动时粒子会被吸引/排斥
  - 鼠标悬停处粒子会发光增强
  - 点击屏幕产生粒子爆炸/波纹效果
  - 滚轮可缩放粒子密度
- 粒子颜色随鼠标位置动态变化（青→品红渐变）
- 整体作为背景层，低透明度不干扰内容阅读

## 四、互动背景技术实现

### 粒子网络系统（Particle Network）
**文件：** 内联在每个HTML页面的 `<script>` 中，或抽离为 `bg-particles.js`

**核心功能：**
1. **Canvas 2D渲染** - 全屏背景画布，z-index: -1
2. **粒子系统** - 80-120个粒子，随机位置/速度
3. **连线逻辑** - 粒子距离 < 150px 时绘制连线，距离越近线越粗越亮
4. **鼠标交互**：
   - `mousemove`：粒子受鼠标引力影响（反向也可，排斥力）
   - `mousedown`：在点击位置生成爆炸粒子（临时增加粒子）
   - `mouseenter`：粒子整体活跃度提升
5. **视觉效果**：
   - 粒子发光（shadowBlur）
   - 连线渐变颜色
   - 低透明度（0.15-0.25）确保不干扰阅读

## 七、风险与注意事项

1. **性能**：粒子数量需控制，移动端自动减少粒子数（如60个）
2. **可读性**：背景透明度需精确控制，确保文字清晰可读
3. **功能保留**：所有现有交互功能（留言、Markdown编辑等）必须保留且正常工作
4. **SEO底线**：基础meta标签（title, description）保留，仅移除过度优化内容
5. **渐进增强**：粒子背景在低端设备上可降级为静态背景
